============================
# Student Management System
============================

## Project Overview:
--------------------

Student Management System is a SQL-based database project used to manage student information, course details, and enrollments.


This project demonstrates the use of SQL concepts such as:
- Database Creation
- Table Creation
- Primary Key
- Foreign Key
- Insert
- Update
- Delete
- Select
- Joins
- Aggregate Functions
- Views

---

## Database Name:

StudentManagementSystem

---

## Tables Used:

### 1. Students
Stores student details.

Columns:
- student_id
- student_name
- age
- gender
- city

### 2. Courses:
Stores course information.

Columns:
- course_id
- course_name
- duration_months

### 3. Enrollments:
Stores student-course enrollment details.

Columns:
- enrollment_id
- student_id
- course_id
